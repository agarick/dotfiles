| =W-S-bksp= | | *close* window |
| =W-= *num* | =1= / =2= / =3= / =4= | *focus nth* desktop |
| =W-A-= *num* | =1= / =2= / =3= / =4= | *move* window to *nth* desktop |
| =W-c= | | *move* window to *center* |
| =W-= *dir* | =h= / =j= / =k= / =l= / =y= / =u= / =b= / =n= | *move* window to *edge west* / *s* / *n* / *e* / *nw* / *ne* / *sw* / *se* |
